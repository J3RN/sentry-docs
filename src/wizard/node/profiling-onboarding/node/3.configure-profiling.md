#### Configure Profiling

Add the `profilesSampleRate` option to your SDK config.

```javascript
import * as Sentry from "@sentry/node";
import "@sentry/tracing";

Sentry.init({
  // ... SDK config
  profilesSampleRate: 1.0,
});
```
